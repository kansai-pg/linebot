# linebot
勉強がてら作成したもの現在Beta

※ベータ版なのであんまり信用しないでください  
何かあっても僕は知りません。  

使い方

|   コマンド         |           説明 |
|:--------------|----------------:|
| 登録              | 部屋名登録 |
| 日時設定              | 通知頻度登録 or 変更 |
| 時刻設定              | 通知時刻変更 |
| 登録一覧                 | 登録部屋一覧表示 |
| 削除                  | 登録部屋削除 |

![lineqr](python-linebot/img/lineqr.png)

statusコード (自分用メモ)
|   状況         |           コード |
|:--------------|----------------:|
| 無し                       | 0 |
| 部屋名入力待機              | 1 |
| 時間入力待機                | 2 |
| グループ当番入力待機         | 3 |
| 頻度入力待機                | 4 |
| 通知時刻変更対象部屋入力待機  | 5 |
| 通知返答待ち                | 6 |
| 非アクティブアカウント返答待ち| 7 |
